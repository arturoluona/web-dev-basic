### Ordered list

<img src="https://user-images.githubusercontent.com/56488686/162630414-6f9044b9-9f6f-413f-91fb-891557f675b0.png" alt="ordered list" width="300">

Make this lists with the following labels
```HTML
    <!-- ordered list -->
    <ol>
        <li></li>
    </ol>

    <!-- ordered list start -->
    <ol start="5">
        <li></li>
    </ol>
    
    <!-- ordered list type -->
    <ol type="A">
        <li></li>
    </ol>
```
